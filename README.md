
# 🌲 台東卑南鄉三塊厝土地租賃專案 (MVP)

這是一個基於 **GitHub Pages** 的靜態土地租賃資訊網頁。專門為位於「台東縣卑南鄉三塊厝段」的農地所設計，旨在透過透明的資訊揭露（包含現況照片、租賃條款與地理位置），降低與潛在承租人的溝通成本。

## 📍 土地概況

* **地點**：台東縣卑南鄉三塊厝段 (鄰近東 38 鄉道)。
* **面積**：約 620 坪 (2052.05 平方公尺)。
* **使用分區**：山坡地保育區 / 農牧用地。
* **核心特色**：首 2-3 年**免租金**（整理期補貼），適合長期投入之果樹種植或休閒農場規劃。

---

## 🚀 技術架構 (資管開發筆記)

本專案採用 **MVP (Minimum Viable Product)** 模式開發，並維持「無伺服器、零成本、高效能」的原則：

* **前端框架**：純 HTML5 / CSS3 (無 JavaScript 依賴，確保極速讀取)。
* **響應式設計**：使用 **CSS Grid** 實作 1.5fr + 1fr 佈局，完美支援電腦與手機觀看。
* **互動邏輯**：採用 **純 CSS 頁籤 (Tabs)** 技術，利用 `radio input` 實現頁面切換。
* **部署平台**：GitHub Pages。
* **社交優化**：整合 **Open Graph (OG) 標籤**，優化 LINE 與 Facebook 分享時的預覽圖與文案顯示。

## 📝 備忘錄 (給自己的提醒)

* **圖片路徑**：務必使用 GitHub Pages 的 `https` 絕對路徑，避免 FB 爬蟲抓取失敗。

* **溝通重點**：洽談時務必確認承租人對「山坡地保育區」法規的了解程度。
